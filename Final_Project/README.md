# Car Price Prediction using Machine Learning

## Description

The objective of this project is to predict the price of a car based on various features such as engine size, horsepower, mileage, and other specifications. A supervised machine learning approach is used to build a regression model that estimates car prices accurately.

---

## Dataset

* **Source:** Provided dataset (used in notebook)
* **Description of Data:**
  The dataset contains multiple features related to cars, such as:

  * Engine size
  * Horsepower
  * Fuel type
  * Mileage
  * Number of doors
  * Other technical and categorical attributes

  The target variable is the **car price**.

---

## Steps Performed

### 1. Data Cleaning

* Handled missing or null values
* Converted categorical variables into numerical format (encoding)
* Removed irrelevant or duplicate data

---

### 2. Exploratory Data Analysis (EDA)

* Analyzed relationships between features and price
* Used summary statistics to understand data distribution
* Identified important features influencing car price

---

### 3. Visualization

* Scatter plots to observe feature relationships
* Correlation heatmap to identify strong predictors
* Distribution plots for understanding data spread

---

### 4. Model Building

* Split dataset into training and testing sets
* Applied regression model (e.g., Linear Regression)
* Trained model on training data
* Predicted prices on test data

---

## Results

### Key Findings:

* Certain features like engine size and horsepower strongly influence price
* Model is able to predict prices with reasonable accuracy
* Feature relationships help in understanding pricing patterns

### Metrics:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* (Optional: R² Score if used in notebook)
