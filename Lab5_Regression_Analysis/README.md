# Lab 5: Regression Analysis

## Description
This lab focuses on implementing regression techniques to analyze relationships between variables and make predictions based on data.

## Objective
The objective of this lab is to understand how regression models work and how they can be used to predict continuous values.

## Dataset
- Boston Housing Dataset  
- Contains numerical features and a target variable (MEDV)

## Steps Performed
1. Imported necessary libraries (Pandas, NumPy, Matplotlib, Scikit-learn)  
2. Loaded and explored the dataset  
3. Performed data preprocessing  
4. Visualized relationships between variables  
5. Applied regression models (Linear Regression, Decision Tree)  
6. Evaluated model performance  

## Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

## Results
- Built regression models successfully  
- Predicted housing prices based on input features  
- Observed relationships between independent and dependent variables  

## Conclusion
Regression models help in understanding data trends and predicting future values effectively.
